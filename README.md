## Description du projet

le projet a été de devopper un site web pour le studio de dévloppement de jeux vidéos indépendant, La Team Cherry, qui consiste en une plateforme reserver a l'entreprise pour covoiturer entre eux mais aussi pour mettre en location des moyens de locomotions tel que des velos ou des trotinettes, pour encourager aux déplacements plus vert.
Notre site contient une page d'acceuil, qui encourage et explique les bienfait de covoiturage. il posséede une page de connexion pour ses utilisateur, afin qu'il puissent reserver ou proposer des crenaux de location ou covoiturage, avec des avis sur les conducteurs.
